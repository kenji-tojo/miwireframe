# MiWireframe

Wireframe mesh rendering examples using [Mitsuba renderer](https://www.mitsuba-renderer.org/).


## Tested environment
- Python 3.12.5

## Dependencies
First, please run
```
pip3 install -r ./requirements.txt
```
to install dependencies.

## Installation
Then, run
```
pip3 install .
```
to build the utility module.

## Running the code
Now, you are ready to run
```
python ./example.py
```
